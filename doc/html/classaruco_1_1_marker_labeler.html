Parameters
dict_typetype of dictionary
error_correction_ratesome dictionaries are susceptible of error correction. These params specify the correction rate: 0 means no correction at all, 1 means full correction (maximum correction bits = (tau - 1) / 2, tau = predefined minimum intermarker distance).

If you want correction capabilities and not sure how much, use 0.5 in this parameter
