\begin{DoxyCompactList}\small\item\em Finds the two possible poses of a square planar object given its four corner correspondences in an image using I\+P\+PE. These poses are sorted so that the first one is the one with the lowest reprojection error. The second pose is needed if the problem is ambiguous. The problem is ambiguous when the projection of the model is close to affine, which in practice happens if it is small or viewed from a large distance. In these cases there are two pose solutions that can correctly align the correspondences (up to noise), so it is impossible to select the right one from just the reprojection error. I\+P\+PE gives you both the solutions, rather than just a single solution (which in ambiguous cases would be wrong 50\% of the time). Geometrically, the two poses roughly correspond to a reflection of the model about a plane whose normal passes through the line-\/of-\/sight from the camera center to the model\textquotesingle{}s center. {\footnotesize\ttfamily void aruco\+::solve\+Pose\+Of\+Centred\+Square (\begin{DoxyParamCaption}\item[{float}]{square\+Length,  }\item[{cv\+::\+Input\+Array}]{image\+Points,  }\item[{cv\+::\+Input\+Array}]{camera\+Matrix,  }\item[{cv\+::\+Input\+Array}]{dist\+Coeffs,  }\item[{cv\+::\+Output\+Array}]{\+\_\+rvec1,  }\item[{cv\+::\+Output\+Array}]{\+\_\+tvec1,  }\item[{float \&}]{reproj\+Err1,  }\item[{cv\+::\+Output\+Array}]{\+\_\+rvec2,  }\item[{cv\+::\+Output\+Array}]{\+\_\+tvec2,  }\item[{float \&}]{reproj\+Err2 }\end{DoxyParamCaption})}



Finds the two possible poses of a square planar object given its four corner correspondences in an image using I\+P\+PE. These poses are sorted so that the first one is the one with the lowest reprojection error. The second pose is needed if the problem is ambiguous. The problem is ambiguous when the projection of the model is close to affine, which in practice happens if it is small or viewed from a large distance. In these cases there are two pose solutions that can correctly align the correspondences (up to noise), so it is impossible to select the right one from just the reprojection error. I\+P\+PE gives you both the solutions, rather than just a single solution (which in ambiguous cases would be wrong 50\% of the time). Geometrically, the two poses roughly correspond to a reflection of the model about a plane whose normal passes through the line-\/of-\/sight from the camera center to the model\textquotesingle{}s center. For more details about these ambiguities, please refer to the I\+P\+PE paper. 

It is possible to reject the second pose if its reprojection error is significantly worse than the first pose. The correct way to do this is with a likelihood ratio test (\href{https://en.wikipedia.org/wiki/Likelihood-ratio_test}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Likelihood-\/ratio\+\_\+test}}).
